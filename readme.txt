== Description ==

A (soon to be) collection of Gutenberg blocks specifically designed to deliver images in an optimised manner.

== Installation ==

Via WordPress:
1. Upload the plugin files to the `/wp-content/plugins/lizardberg` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

Via Composer:
1. Add the following repo to your `composer.json` file:
```json
{
    "type": "composer",
    "url": "https://composer.jackwhitworth.com"
}
```
2. Run: `composer require jmwhitworth/lizardberg`
